A terminal app to perform report card related functions for teachers.

About

The Student Report Card App is a console-based application designed to streamline the process of generating student report cards. Developed using the C++ programming language, this system is user-friendly and can be easily modified to suit specific needs. The primary purpose of this system is to facilitate the distribution of grade records to each student, making it easier for teachers to compute the General Average for student grades. It is particularly beneficial for students taking computer-related courses, as it provides new techniques in C++ programming.

Doctor's Appointment System

A terminal app to make your doctor appointments.

About

The Doctor's Appointment System is a console-based application designed to simulate the appointment system of any hospital. Developed using the C++ programming language, this system is designed to maintain records even if the application is closed. The primary purpose of this system is to manage doctor appointments. It provides a list of available slots to customers, allowing them to choose any slot that is not already booked. This system ensures that customers cannot book a slot that has already been reserved, thereby preventing double bookings and ensuring efficient management of doctor's schedules.
